Fresh Market Event Creator

Documentation for installing and using The Fresh Market Event Creator application.

Contents

  • Installation
  • Usage

Installation

To install the application download and run the EventProcessor.msi installer on your Windows workstation. You do not need to have administrator privileges to install or run the application.

The first step of installation is to accept the license agreement by clicking the checkbox and then clicking the Install button.

The installation process will then execute and add the application to your workstation.

The installation will finish, click the Finish button to exit from the installer

Usage

Run the application by selecting the Event Creator application from your Windows Startmenu.

At the login screen enter a valid set of ExactTarget credentials for the account

Either type in the location of an event data spreadsheet or press the ... button to browse for one. Be sure to not have the spreadsheet open in Excel or any other program.

Enter an email address for the events to use as an organizer contact.

Press the Create Events button which will:

  • Read the events from the rows in the spreadsheet
  • Create ICS files for each event as Portfolio items
    • The ICS files will be stored in subfolders named for the event date underneath the EventCalendar folder
  • Create a row in the Data Extensions > EventCalendar > Events data extension that contains the event details and the URL to the ICS portfolio item

When the spreadsheet has been processed you will receive a notification of how many events were created and prompting if you want to export an activity report.

If you select Yes and export the report it is a text file that gives the details of the events that were created.

You can now see the ICS files in the subfolders underneath the EventCalendar folder.

Exit the application by clicking on the X in the upper right hand corner of the window.

Now all emails that are built from the Template: My Templates > Additional Emails > 2014 > Fresh Flavors Responsive Email template will automatically pull in the details of the next upcoming event at the subscribers preferred store.
